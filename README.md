# 🚀 Cultus Distributed KV Store (Raft Simulation)

A lightweight distributed key-value store built using Node.js that simulates **Raft consensus**, **leader election**, **log replication**, and **network partition handling**.

---

## 🧠 Key Features

- 👑 Leader Election (Raft-style simulation)
- 📦 Distributed Key-Value Store
- 🔁 Log Replication across nodes
- 🌐 Network Partition Simulation
- 💾 Persistence using JSON storage
- ⚡ REST API using Express
